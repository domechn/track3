✨ Features

Everything you need to track your portfolio

From exchange balances to on-chain wallets, Track3 brings all your crypto holdings into one private, beautiful dashboard.

🔒

100% Local & Encrypted

All data is stored locally in an encrypted SQLite database. No servers, no cloud — your API keys and wallet addresses never leave your machine.

📈

Rich Analytics & Charts

Dozens of charts including asset proportions, PnL curves, top coins ranking, wallet breakdowns, historical performance, and more.

🔗

CEX & Web3 in One Place

Connect centralized exchanges via read-only API keys and track on-chain wallets (BTC, ETH, SOL, DOGE, TON, SUI) simultaneously.

💰

Profit & Loss Tracking

Track your realized and unrealized PnL with cost-basis calculations, asset change history, and ATH (all-time-high) monitoring.

📦

Backup & Restore

Export your data to local files and import on other devices. Sync between machines using iCloud, Dropbox, or any cloud drive.

⛓

No Wallet Signature Required

Track3 only reads public on-chain data and exchange balances. It never requires transaction signing or fund access permissions.
